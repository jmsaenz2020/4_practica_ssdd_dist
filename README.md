# Practica 4

### Instrucciones para compilar y ejecutar el programa
Dentro del directorio del repositorio, ejecute los siguientes comandos:
```
$ cd src/
$ go build
$ ./4_practica_ssdd_dist
```

### Instrucciones para ejecutar los tests
Dentro del directorio del repositorio, ejecute los siguientes comandos:
```
$ cd src/
$ go test ./...
```
